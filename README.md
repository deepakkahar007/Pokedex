## ðŸ¤¸ Quick Start

1. clone Repo

```bash
git clone https://github.com/deepakkahar007/pokedex.git
```

2. go to folder

```bash
cd pokedex
```

3. open folder in IDE

4. install all the depeendency

- for client

```bash
cd client
npm i
```

- for server

```bash
npm i
```

5. start the server

```bash
cd server
npx json-server db.json
```

6. start client project

```bash
cd client
npm run dev
```
